# Product Vision

Trade Navigator should evolve from a portfolio viewer into a portfolio operating system.

Core goals:
- sleeve-based portfolio management
- risk-first monitoring
- rule-driven decisions
- action queues
- suggested manual orders
- opportunity discovery
- future IBKR integration
- manual support for Wio and Expat sleeves

Core design principle:
The app should answer:
1. what matters
2. what changed
3. what is risky
4. what needs action
5. what to do next

Primary entities:
- Portfolio
- Sleeve
- Position
- Rule
- Alert
- Action Item
- Order Suggestion
- Watchlist Item
- Thesis
- Scenario

Immediate priorities:
- cleaner architecture
- decision-first home screen
- sleeve-aware navigation
- action center
- order suggestion v1
